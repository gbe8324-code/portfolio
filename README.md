# GBE Portfolio

HR 전문가 포트폴리오 웹사이트

## 📋 소개

건설업, 법무, 리테일 등 다양한 산업에서 인사 운영 경험을 가진 HR 전문가의 포트폴리오 사이트입니다.

## 🎯 주요 기능

- **경력사항**: 상세한 경력 이력 및 주요 성과 소개
- **학력 및 자격**: 학력 정보 및 자격증 표시
- **핵심 역량**: 주요 역량 시각화
- **반응형 디자인**: 모든 기기에서 최적화된 디자인
- **부드러운 스크롤**: 네비게이션을 통한 부드러운 이동

## 📁 파일 구조

```
.
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 상호작용
└── README.md       # 이 파일
```

## 🚀 시작하기

1. 저장소 클론
```bash
git clone https://github.com/gbe8324-code/portfolio.git
cd portfolio
```

2. 로컬 서버에서 열기
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (http-server 설치 필요)
http-server
```

3. 브라우저에서 `http://localhost:8000` 접속
